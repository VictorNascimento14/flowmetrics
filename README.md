## Sobre o Projeto

O Flowmetrics foi desenvolvido para transformar dados brutos em informações estratégicas, utilizando técnicas avançadas de análise de dados e inteligência artificial. A plataforma automatiza a coleta, tratamento e visualização de métricas de produtividade, permitindo que empresas tomem decisões baseadas em dados reais e confiáveis.

Como Analista de Dados, atuei em todo o ciclo de dados: desde a integração e limpeza dos dados, passando pela análise exploratória, até a construção de dashboards interativos e relatórios automatizados. Utilizei IA para gerar insights preditivos e identificar padrões relevantes para o negócio.

## Principais Funcionalidades

- Coleta automatizada e análise de dados de produtividade
- Visualização interativa e customizável de métricas
- Geração automática de relatórios gerenciais
- Integração com APIs de IA (Gemini) para insights avançados
- Dashboards dinâmicos para acompanhamento em tempo real
- Detecção de padrões e tendências usando algoritmos de IA

## Como Executar Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   ```
   npm install
   ```
2. Configure sua chave da Gemini API em `.env.local`:
   ```
   GEMINI_API_KEY=SuaChaveAqui
   ```
3. Execute o app:
   ```
   npm run dev
   ```
